Приложение получает на вход строку с адресом Web-страницы, сохраняет ее на жесткий диск компьютера, подсчитывает количество уникальных слов на странице и выводит статистику по словам в консоль

------------------

Для запуска приложения необходимо открыть командную строку в папке с исполняемым модулем и прописать команду

------------------

java -jar SimbirSoftTest-0.1-shaded.jar --folder='abc' --type='abc'--db.use=true

------------------

указывать необходимо без '' ковычек

Параметры:

------------------

--folder - путь на жестком диске, в который сохранится скачанная Web-страница под именем temp.html

--type - тип вводимой ссылки(возможны 2 варианта для корректной работы приложения:
	url-если вводимая строка url-ссылка на ресурс
	file - если вводимая строка путь к файлу с разрешением .html (используется для тестов)


--db.use - значения true или false (режим, обозначающий использование базы данных, true - статистика сохранится в базу данных, false - база данных не будет использоваться

------------------

после данной команды необходимо ввести ссылку на ресурс, который нужно проанализировать

------------------

Eсли --db.use=true:

необходимо в своей базе данных выполнить скрипт создания таблицы

create table statistic
(
    id         bigserial not null
        constraint statistic_pk
            primary key,
    site       varchar,
    word       varchar,
    word_count integer
);

далее программа попросит ввести 4 значения

db.driver - указывается драйвер для используемой бд (например: org.postgresql.Driver)
db.url - указывается урл базы данных
db.user - имя пользователя базы даных
db.password - пароль для пользователя базы данных
